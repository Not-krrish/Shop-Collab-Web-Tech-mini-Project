🏪 Local Shop Collaboration Finder : "NextLocal"

📌 Problem Statement

Small local businesses face difficulty in discovering nearby shops and establishing collaborations for mutual growth, promotions, and customer sharing. There is no simple platform available to explore and connect with nearby businesses.
